Close permissive

The rung no 50 or 51 trip is written NOT(50 OR 51). De Morgan makes it equal to (NOT 50) AND (NOT 51); the interpreter shows both forms agree for the current inputs.

The 86 latch, on its own

A lockout relay is a latch. Set it with a trip, and it HOLDS with no input. Reset dominant means a reset always wins, so an operator can always clear it. The output survives a reboot, which is what a hand reset lockout does.
