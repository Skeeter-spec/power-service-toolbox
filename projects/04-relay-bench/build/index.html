Relay bench

Seven ANSI protective elements as configurable simulated relays. Set the parameters, inject a signal, watch the element decide. Each one is built against a published document and names it, and the time-overcurrent element runs on this toolbox's own coordination engine rather than a copy of its math.

Educational. Not for field use. These reproduce published behaviour so it can be explored. Nothing here is an engineered protection setting, a substitute for the manufacturer's instructions, or a device you would trust to trip a real breaker.
